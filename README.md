**ДЗ 98. Card.jsx**

Реалізуйте компонент <Card> так, щоб можна було складати таку структуру 

```
<Card>
  <Card.Body>
    <Card.Title>Title</Card.Title>
    <Card.Text>Text</Card.Text>
  </Card.Body>
</Card>
```
Отриманий HTML
```
<div class="card">
    <div class="card-body">
        <h4 class="card-title">Title</h4>
        <p class="card-text">Text</p>
    </div>
</div>
```
